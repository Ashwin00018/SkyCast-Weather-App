# SkyCast - Weather Application

SkyCast is a modern and responsive weather application built using HTML, CSS, and JavaScript. It provides real-time weather information for any city using the WeatherAPI. The application features dynamic weather backgrounds, rain animations, and a clean user interface for an engaging user experience.

---

## Features

- Search weather for any city
- Real-time temperature
- 7-Day Weather Forecast
- Sunrise & Sunset timings
- Wind Speed
- Humidity
- Air Quality Status
- Visibility
- UV Index
- Dynamic weather backgrounds
- Rain animation
- Loading animation
- Fully Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI

---

## How to Run

- Clone or download this repository.
- Open the project folder.
- Open `index.html` in your browser.
- Search for any city to view weather information.
