About SkyCast

SkyCast is a modern weather application that provides real-time weather information for any city using the WeatherAPI. It displays current weather, temperature, humidity, wind speed, pressure, visibility, UV index, sunrise & sunset, and a 7-day weather forecast. The app also features dynamic backgrounds and rain animations that change according to the current weather conditions, creating a more interactive user experience.
